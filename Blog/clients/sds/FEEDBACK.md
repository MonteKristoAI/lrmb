## [2026-04-07] — Email thread `19d683fab844d4f5` (Fwd: ACTION: Marketing Campaign - Site Revamp)

**Context:** Christopher's line-by-line revision of the Brand Voice Manifesto v2, which produced Manifesto v4.

**Rules generated:**

1. **Ban "interoperable" / "interoperability".** Too complicated for the audience. Use **"universally compatible" / "universal compatibility"** instead. Global find-and-replace.
2. **Ban "Subsurface Intelligence".** Buzzword that won't land with customers. May be usable in the future for a cloud visualization product, not for current launch.
3. **Ban "Switzerland of Data".** No meaning in this context.
4. **Ban "Data Sovereignty" framing.** Not a differentiator. Customers care about easy data access (streaming, exports, manipulation), not philosophical "ownership" claims.
5. **Ban "Your data is yours".** Same critique. Drop the phrasing.
6. **Ban "guided workflows / guided automation / intelligent automation / supervised automation".** Warrior does NOT automate setup. Only automation in the product is Auto-Perf firing. Use **"standardized interfaces"** instead.
7. **Ban tractor-conveyed / high-power direct connect** claims. Planned but NOT shipping. Cannot imply features the platform doesn't have.
8. **Ban "proprietary formats"** as an abstract term. Too vague. Use **"closed-system limitations"** or **"black-box ecosystems"**.

**How to apply:** Run a grep against `clients/sds/copy/banned-vocab.md` before publishing any post. Zero tolerance.
