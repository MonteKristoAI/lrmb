# SDS / Warrior — Authoritative Source Registry

**Purpose:** Every SDS blog post pulls external citations from this file. Do NOT cite sources that are not on this list unless they pass a tier 1-3 test (see bottom). The list prioritizes sources that maximize E-E-A-T, AI citation signal, and credibility with wireline professionals.

**Rule:** Minimum 5 Tier 1-3 sources per post. 8-10 for pillar posts.

## Tier 1 — Peer-Reviewed / Standards / Government (Highest Authority)

### Standards bodies

**American Petroleum Institute (API)**
- Home: [api.org](https://www.api.org/)
- Relevant documents:
  - **API RP 67 — Recommended Practice for Oilfield Explosives Safety** — the perforating safety bible. Editions: 1st 1994, 2nd 2007, 3rd 2015, **4th edition in development** with RF-distance rules. Listed at [GlobalSpec](https://standards.globalspec.com/std/13492745/api-rp-67).
  - **API RP 66 — Digital Log Interchange Standard (DLIS)** — the binary wireline data format standard.
  - **API RP 19B** — perforator performance evaluation.
  - **API RP 19D** — measurement of explosive energetic materials.
  - **API Spec Q1** — quality management system.
- Citation format: "API RP 67, Recommended Practice for Oilfield Explosives Safety (2015)"

**ISO (International Organization for Standardization)**
- Home: [iso.org](https://www.iso.org/)
- Relevant:
  - **ISO 10417** — subsurface safety valves (relevant to wireline intervention)
  - **ISO 13503** — completion fluids
  - **ISO 15156** — materials for H2S environments (sour service wireline)

**ASME (American Society of Mechanical Engineers)**
- Home: [asme.org](https://www.asme.org/)
- Relevant: Boiler and Pressure Vessel Code Section VIII (lubricators, wireline pressure control)

## Vetting Test (for sources not on this list)

Before citing a source that is NOT on this list, run this test:

1. **Authorship** — Is a named author listed? (Must be yes for Tier 1-3)
2. **Credentials** — Does the author have a verifiable professional or academic affiliation?
3. **Publication age** — Is it published within 5 years? (Exceptions: API RPs, LAS standards, historical references)
4. **Peer review or editorial review** — Is it peer-reviewed, editorially reviewed, or first-party from a known authority?
5. **Primary vs secondary** — Is this the original source, or a summary? Prefer primary.
6. **Conflict of interest** — Does the author have a commercial interest in the claim? (Tier 4 tag if yes)
7. **URL stability** — Is the URL likely to exist in 3 years? (Avoid press release aggregators)

**Score:** 6+ green → Tier 2 or 3 citation acceptable. 5 or fewer → do not cite.

## Citation Rules (enforced by blog-factcheck skill)

1. Every statistic must have a year inline ("in 2025", "as of 2015", etc.)
2. Every statistic must link to a source via `<a href="..." target="_blank" rel="noopener noreferrer">`
3. Every citation must have been verified within 3 months of the post publication
4. No Wikipedia citations except as a fallback when a primary source is unavailable
5. No Reddit or forum citations as primary evidence (OK for quoted voice only)
6. No AI-generated content can be cited
7. Run `/blog factcheck` before any post is marked done
