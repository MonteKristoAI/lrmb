## Reading This File

Every keyword is tagged with:
- **Volume** (monthly, US) — from DataForSEO live
- **Tier** — T1 (500+), T2 (100-499), T3 (10-99), T4 (0) — see volume tiers below
- **Cluster** — which topic cluster it belongs to (A-H)
- **Intent** — Informational / Commercial / Transactional / Navigational
- **Priority** — P0 (launch post target) / P1 (post-launch) / P2 (authority fill) / T4 (no dedicated post, weave into body copy)

**Strategic lens:** this niche is a citation play, not a volume play. Do not dismiss 0-volume keywords — they still drive AI citation pickup and LinkedIn discovery. A post targeting "warrior universal panel" at 0/mo can still become the #1 AI citation for "what is a universal wireline panel?" and generate dozens of qualified leads per month.
