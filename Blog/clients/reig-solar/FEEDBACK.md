## [2026-04-01] — Non-negotiable SEO + quality rules (direct instruction)

**These are hard gates. No post ships without satisfying every item.**

- **SEO Title**: strictly 50–60 characters. Minimum 50, maximum 60. Count before finalizing. No annotation, no estimation — use an exact character count.
- **Meta description**: strictly **< 155 characters**. Hard ceiling. Target 145–154. Count before finalizing.
- **Focus keyword** must be present in: SEO title, meta description, H1, first paragraph, ≥30% of all H2+H3 headings, at least 1 image alt attribute.
- **SVG charts**: text labels must NEVER overflow their container. Before saving any SVG, verify that every text element fits. For bar charts, if bar width < (character_count × 7px), the label MUST be placed outside the bar, not inside.
- **External link verification**: fetch EVERY external link via WebFetch before delivery. Standards body URLs (IEEE, NERC, IEC, NIST) frequently change. Never use internal document ID URLs for IEEE (e.g. `/ieee/1815/5925/` is wrong) — always use the stable format: `standards.ieee.org/standard/XXXX-YYYY.html`. Zero broken or wrong external links allowed at delivery.
- **Quality parity with BreathMastery**: REIG posts must meet the same overall quality standard as BreathMastery — required elements checklist, keyword density, transition word coverage, human burstiness, FAQ length (40–60 words each). These are not optional.
